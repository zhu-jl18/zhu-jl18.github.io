Posted on 2021-03-28 Edited on 2021-04-29 In Configuration

Reference:

  • 简单配置与美化Powershell和Terminal
  • Windows Terminal 美化实例

At the beginning, transform Powersell with oh-my-posh:

Import-Module posh-git
Import-Module oh-my-posh
Set-Theme paradox
$ThemeSettings.PromptSymbols.VirtualEnvSymbol = "ENV"
# Chocolatey profile
$ChocolateyProfile = "$env:ChocolateyInstall\helpers\chocolateyProfile.psm1"
if (Test-Path($ChocolateyProfile)) {
  Import-Module "$ChocolateyProfile"
}
