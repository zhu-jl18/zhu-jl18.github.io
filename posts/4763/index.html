自动续传流程：

  1. 检测截断: Worker 实时监控响应状态
  2. 保存上下文: 保留原始请求的上下文信息
  3. 构造续传请求: 基于截断点构造新的 API 调用
  4. 智能拼接: 将续传内容与原内容无缝拼接
  5. 返回完整响应: 返回给 GPT-Load 再转发给 Cherry Studio
