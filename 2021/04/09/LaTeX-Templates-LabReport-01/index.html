LaTeX-Templates-LabReport-01

Posted on 2021-04-09 Edited on 2021-04-29 In LaTeX

A simple but useful template for common lab-report.

It has the following features:

  • different fonts
  • multipic and pic
  • 3-lines table
  • Ref
  • Appendix

Code
